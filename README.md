# Dharma-Yoga
A responsive React landing page for a yoga studio, built with React and focused on clean design, usability, and mobile responsiveness.

Live demo: https://dharmastudio.netlify.app/

## Features

- Modern, minimal UI design
- Fully responsive layout (desktop, mobile)
- Custom hero section and content blocks
- Reusable React components
- Optimized static assets

## Tech Stack

- React
- JavaScript
- HTML5 & CSS3
- Yarn

## How to launch locally

- Prerequisites: Node.js, Yarn
- Installation:
```bash
git clone https://github.com/spiriiito/dharma-yoga.git
cd dharma-yoga
yarn install
yarn start
```
The app will be available at http://localhost:3000.
